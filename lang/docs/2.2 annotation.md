# Java Annotation

## Basic Concepts
 - 注解是编译期写入的元数据，虽然可以在运行期通过反射得到注解的信息，但是注解在运行期是不可改变的（或者说没有改变的意义）
 - 因此注解和对象无关，这个概念是讨论Java注解的基础。
 - 类，方法，变量，参数和包都可以被注解
   * 类的注解，每个由此类生成的对象共享类的注解
   * 类的方法的注解，判断类中定义的某个方法所否被注解
   * 类的变量（任意地方出现的变量，局部变量如何反射呢？）
   * 参数
